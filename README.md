`Let's Block Chain` provides a simple block-chain implementation and simple, elegant command line tool write with Golang.

## Prerequisites
- Golang 1.12.1
- GNU Make

## Milestone

- [x] Basic Prototype
- [x] PoW with Hashcash
- [x] Block Chain Persistence
